Disclaimer & Important Notes

            

                
This tool is an unofficial helper application designed to assist in visualizing and editing XML skin files for chipsynth synthesizers by Plogue.

                 
It attempts to interpret the XML structure and render a visual representation. However, it is not a complete emulation of the chipsynth engine's GUI rendering. Discrepancies may exist.

                 
Always back up your original skin files before making changes with this tool. Plogue is not responsible for any issues arising from the use of this editor.

                 
This tool primarily focuses on layout and visual properties. Complex interactions, dynamic content, and some specific control behaviors might not be fully represented or editable.

                 
Use at your own risk. The developers of this tool are not liable for any data loss or corruption.
